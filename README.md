# 📊 Interactive Sales Performance Dashboard

## 📝 Project Overview
This project features a comprehensive **Sales Analytics Dashboard** built in Microsoft Excel. It is designed to transform raw transactional data into actionable business insights, enabling stakeholders to track performance across regions, product categories, and time periods. The dashboard serves as a central tool for monitoring key sales KPIs and identifying trends in the supply chain.

## 🚀 Key Features
*   **Executive Summary KPIs:** Real-time tracking of critical metrics, including **Total Sales** (~$12.9M), Total Quantity, and Order Counts.
*   **Dynamic Filtering:** Interactive slicers for **Order Date** and **Ship Method**, allowing users to isolate specific data segments instantly.
*   **Multi-Dimensional Visualization:**
    *   **Territory Analysis:** Bar chart comparing sales performance across key regions (Australia, Canada, France, etc.).
    *   **Category Breakdown:** Pie chart visualizing revenue distribution by product type (Bikes, Accessories, Clothing).
*   **Granular Data Views:** Detailed Pivot Tables providing a drill-down capability into specific product models, sub-categories, and regional sales figures ("Total Due").

## 🛠️ Tools & Technologies
*   **Microsoft Excel**
*   **Pivot Tables & Pivot Charts**
*   **Slicers & Timelines**
*   **Conditional Formatting**

## 💡 Insights
This dashboard allows for rapid analysis of:
*   Which territories are underperforming versus overperforming.
*   The revenue contribution split between high-value items (Bikes) and consumables (Accessories).
*   The impact of shipping methods on total sales volume.
